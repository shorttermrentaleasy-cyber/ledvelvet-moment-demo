# LedVelvet MVP (Local Demo)

Questa cartella contiene una demo locale (Next.js + Tailwind) con:
- `/moment` → mockup stile *Cercle Moment* (hero video + card evento video, membership, shop)
- `/demo` → demo alternativa dark
- video hero locale in `public/media/petra_led.mp4`

## Requisiti
- Node.js 18+ (consigliato 20)
- npm (o pnpm/yarn)

## Avvio
```bash
npm install
npm run dev
```

Apri:
- http://localhost:3000
- http://localhost:3000/moment
- http://localhost:3000/demo

## Nota video
Se vuoi cambiare il video hero, sostituisci:
`public/media/petra_led.mp4`
